# C-RISC-V-Embedded-Journey

A project-driven path from C basics to embedded systems through aquaculture-oriented tools.  

## 1. What This Is

This repository is my project-driven journey of learning **C**, **RISC-V / embedded systems**, and **aquaculture-oriented tooling**.

## 2. How To Learn

I do not want to learn C as isolated syntax. I want every concept to become a small runnable loop, and every loop to grow into a real system.

## 3. Where This Goes

The direction is simple: from basics, to records, to tools, to embedded integration. I want to build practical systems that can describe, judge, record, and eventually connect real aquaculture data.
